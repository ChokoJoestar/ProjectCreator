## Install

### With NPM
```sh
# Locally
npm install -D @chokojoestar/project-creator

# Or globally
npm install -g @chokojoestar/project-creator
```

### With YARN
```sh
# Locally
yarn add -D @chokojoestar/project-creator

# Or globally
yarn add -g @chokojoestar/project-creator
```
